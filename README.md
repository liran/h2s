# h2s #

a daemon transport http request to socks proxy

    Usage of ./h2s:
     -addr string
        address listen to (default "127.0.0.1:1081")
      -help
            show this help
      -socks_addr string
            socks server addr (default "127.0.0.1:1080")
      -socks_proto string
            socks protocol type, [socks5|socks4|socks4a] (default "socks5")
